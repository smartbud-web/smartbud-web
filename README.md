# Smart Bud 樂芽 | Premium Baby & Toddler Brands Curator Hong Kong

這是一個專為香港高階母嬰精品選品店 **Smart Bud 樂芽** 設計的響應式電商前端單頁式網頁應用（SPA）。

## 🚀 網頁特點
- **極致的前端架構**：採用純原生 JavaScript 與 CSS 動畫實現流暢的單頁分頁切換（SPA），無須重新整理網頁。
- **美學與科學兼具**：整合 Tailwind CSS 進行高雅的品牌色調（奶油白、嫩芽綠、晨曦金）客製化。
- **國際化支持**：具備繁體中文（ZH-HK）與英文（EN）語系即時切換功能。

## 📁 檔案架構
- `index.html` - 主要網頁結構
- `css/style.css` - 自訂 CSS 翻轉特效與呼吸動畫
- `js/app.js` - Tailwind Config 擴充、購物車與路由邏輯

## 🛠️ 如何在本機執行
1. 將本專案 `git clone` 到本機計算機。
2. 直接連按兩下點開 `index.html`，或者使用 VS Code 的 **Live Server** 套件開啟即可預覽。
